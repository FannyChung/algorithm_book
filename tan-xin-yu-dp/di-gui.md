# 递归

特点：

* 子问题
* 出口（basecase）
* 处理子问题结果
* 不记录子问题的解

## 汉诺塔

把from的N个块移到to上。限制：只能小的块在大的块上面。

思路：

先把N-1个移到mid上——子问题

把第N个（最大的）移到to

把mid上的N-1个移到to——子问题

出口：N=1



复杂度O\(2^N\)



## 打印全部子序列

子序列可以不连续。比如abc的子序列有a、ab、ac等六个。

对于当前字符，可以选择取或者不取（联想到二叉树）。

所以一共有2^N-1个子序列（空除外）。

用递归的方法实现：

```java
f(str,i,before):
if(i==str.length()): print(before);

//否则，可以选择取或者不取
f(str,i+1,before+str(i));
f(str,i+1,before);
```
